- ✨ **1:1 & group chats**

- 📨 **Message read receipts**

- 📷 **Image/file sharing**

## 🚀 Quick Start

### Prerequisites
- Node.js ≥16
- MongoDB Atlas account

### Installation
```bash
# Clone repo
git clone https://github.com/yourusername/chat-app.git
cd chat-app

# Install dependencies
cd backend && npm install
cd ../frontend && npm install

# Configure environment variables
